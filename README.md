## Setup instructions

### 1.  Enter the project source code folder

```sh

cd ./bluemessenger
```

### 2. Install dependencies

```sh
# Install npm dependencies
npm install
react-native link
```
### 3. Install the app on emulator

```
npm run android
```
### 3. Start your app

```
npm start
```
